Few-shot 개념
Few-shot이란?
AI에게"이런 식으로 대답해 줘"라는 패턴을 예시로 보여주어

Zero-shot
예시 없이 바로 질문을 던지는 방식. AI의 기본 지식에 전적으로 의존합니다.
One-shot
단 하나의 예시를 제공하여 대략적인 방향성을 제시합니다.
Few-shot
여러 개의 예시를 통해 패턴을 명확히 학습시킵니다. 가장 안정적이고 품질 높은 결과를 보장합니다.

기본 워크플로우 노드 6종
시작 노드 (Start)
기능
워크플로우의 시작점입니다.
 사용자의 초기 입력 메시지를 받아 다음 단계로 전달합니다. 필수 노드입니다.
종료 노드 (End)
기능
워크플로우의 실행을 마치고 최종 결과를 사용자에게 출력합니다.
 모든 흐름은 이 노드에서 끝납니다.
프롬프트 노드 (Prompt)
기능
LLM에게 구체적인 지시를 내립니다.
 역할(Persona)을 부여하거나 특정 작업을 수행하도록 텍스트를 생성합니다.
스킬 노드 (Skill)
기능
특정한 기능을 수행하는 도구입니다.
 퀴즈 생성, 문법 교정, 학습 플래너 등 미리 정의된 기능을 실행합니다.
지식베이스 노드 (Knowledge)
기능
RAG(검색 증강 생성) 기술을 활용합니다.
 업로드된 문서에서 질문과 관련된 내용을 검색하여 참조합니다.
MCP 도구 노드 (MCP)
기능
외부 서비스와 연동합니다.
 실시간 웹 검색, 계산, 환율 조회 등 외부 데이터를 가져와 처리합니다.

로직 & 고급 노드 5종
분류 노드
Classification
사용자의 입력 의도를 분석하여 적절한 경로로 자동 분기합니다.
활용 예시
"문의 내용이 환불/배송/기타 중 어디에 해당하는지 자동 분류"
조건분기 노드
If / Else
변수 값이나 특정 조건의 충족 여부에 따라 워크플로우 흐름을 제어합니다.
활용 예시
"회원 등급이 VIP인 경우 전담 상담원 봇으로 연결"
변환 노드
Transform
데이터의 포맷을 변경하거나 결과에서 특정 값만 추출하여 가공합니다.
활용 예시
"복잡한 JSON 응답 데이터를 읽기 쉬운 텍스트 리포트로 변환"
상태설정 노드
Set State
대화 문맥이나 사용자 정보를 변수에 저장하고 업데이트합니다.
활용 예시
"사용자의 언어 설정이나 관심사 정보를 기억하여 다음 대화에 반영"
가드레일 노드
Guardrail
입력 및 출력 데이터를 검사하여 부적절한 내용을 필터링하고 보안을 강화합니다.
활용 예시
"응답에 개인정보(전화번호 등)가 포함된 경우 마스킹 처리"

1
워크플로우

노드

3
스킬
재사용 가능한 AI 능력 모듈입니다. 
한 번 만들면 여러 에이전트에서 반복해서 사용할 수 있습니다.
4
세션
에이전트와의 연속 대화 단위입니다. 이전 대화 내용을 기억하며 맥락을 이어갑니다.

비유
연필, 계산기, 사전 같은 도구
업무 매뉴얼, 순서도 같은 절차
관계
에이전트 안에서 MCP 노드로 사용
MCP 도구를 워크플로우의 한 단계로 활용
