# Step 4: MCP 도구

## 기능 소개

MCP(Model Context Protocol)는 **AI 모델이 외부 도구를 호출하는 표준 방식**입니다. AI에게 "손과 발"을 달아주는 기능으로, **12개 카테고리 · 총 83종의 외부 도구**(빌트인 40종 + 외부 서비스 연동 43종)를 제공합니다.

- **핵심 포인트**: AI가 학습된 지식에만 의존하지 않고 실시간 데이터를 조회하거나 도구를 사용하여 더 정확한 답변을 제공합니다.
- **AI 자동 구성 기능**으로 프롬프트 하나만 입력하면 필요한 도구를 자연스럽게 연결해줍니다.

> **프로세스**: 사용자 질문 → AI 모델(LLM) → MCP 표준 프로토콜(중개자) → 외부 도구(날씨 API, 검색 엔진, 계산기 등) → 결과 → 답변

## MCP vs 에이전트 차이

| 구분 | MCP (도구) | 에이전트 (두뇌) |
|------|-----------|----------------|
| **역할** | AI의 손과 발 — 실행 및 데이터 조회 | AI의 두뇌 — 판단 및 지시 |
| **기능** | 외부 시스템/API 연결, 실시간 데이터 확인, 정확한 계산 | 시스템 프롬프트로 페르소나 정의, 작업 단계 계획/추론, 사용자 의도 파악 |
| **비유** | 칼과 불(요리 도구) | 요리사 |

<Note>
요리사(에이전트)가 아무리 뛰어나도 **칼과 불(MCP) 없이는 요리 불가**. 두 가지를 함께 사용할 때 최고의 결과를 얻을 수 있습니다.
</Note>

| **복잡한 수학 계산** | "약 9천 7백만 정도" (부정확) | "정확히 97,406,784" (Calculator 도구 활용) |
| **실시간 환율** | "2023년까지 지식으로 정확한 환율 불가" | "원/달러 환율 1,450.50원 (전일 대비 ▲5.0원)" |

## AI MCP 생성

원하는 MCP 기능을 **자연어로 입력**하면, AI가 플랫폼에서 제공하는 도구를 자동으로 조합하여 MCP를 생성해줍니다.

<Steps>
  <Step title="AI 생성 탭 선택">
    MCP 도구 메뉴에서 **[AI 생성]** 탭을 클릭합니다.
  </Step>

  <Step title="원하는 기능 입력">
    "어떤 MCP를 만들까요?" 입력란에 원하는 기능을 **자연어로 설명**합니다. 하단의 예시(리서치 봇, 데이터 분석, 실시간 날씨 봇, 학습 도우미)를 참고할 수 있습니다.

    <Frame caption="AI MCP 생성 — 자연어 입력 화면">
      <img src="/images/MCP_AI 생성(1) .png" alt="AI MCP 생성 입력 화면" />
    </Frame>
  </Step>

  <Step title="AI로 MCP 생성">
    **[AI로 MCP 생성]** 버튼을 클릭하면 AI가 자동으로 도구를 조합하여 MCP를 구성합니다. 우측의 실시간 미리보기에서 도구 구성과 실행 흐름을 확인할 수 있습니다.

    <Frame caption="AI MCP 생성 — 자동 구성 결과">
      <img src="/images/MCP_AI 생성(2).png" alt="AI MCP 생성 결과" />
    </Frame>
  </Step>

  <Step title="저장">
    구성 내용을 확인한 후 **[저장]** 버튼을 클릭합니다. 필요시 **[다시 생성]**으로 재구성할 수 있습니다. 저장된 MCP는 **[내 MCP 목록]** 탭에서 확인할 수 있습니다.
  </Step>
</Steps>

## 7가지 MCP 템플릿

| 템플릿 | 설명 | 권장 도구 |
|--------|------|----------|
| **학습 도우미** | 수업 내용 이해/학습 보조 | 웹 검색, 계산기, 날짜/시간 |
| **리서치** | 정보 수집과 분석 | 웹 검색, 웹 크롤링, 파일 읽기 |
| **코딩 연습** | 프로그래밍 학습 | 계산기, 웹 검색, 날짜/시간 |
| **글쓰기** | 작문/에세이 작성 | 웹 검색, 파일 읽기 |
| **수학/과학** | 수학 계산/과학 탐구 | 계산기, 웹 검색 |
| **외국어 학습** | 언어 학습/번역 | 웹 검색, 웹 크롤링 |
| **자유 구성** | 나만의 MCP 구성 | 모든 도구 선택 가능 |

## 나만의 MCP 만들기

플랫폼에서 제공하는 도구를 직접 조합하여 나만의 MCP를 만들 수 있습니다.

<Steps>
  <Step title="도구 선택">
    MCP 도구 메뉴에서 **[나만의 MCP 만들기]** 탭을 클릭합니다. 플랫폼 도구 목록에서 원하는 도구를 **체크하여 조합**한 후 **[다음]**을 클릭합니다.

    <Frame caption="나만의 MCP 만들기 — 도구 선택">
      <img src="/images/MCP_나만의 MCP 만들기.png" alt="도구 선택 화면" />
    </Frame>
  </Step>

  <Step title="이름 & 저장">
    **MCP 이름**과 **설명(선택)**을 작성한 후 **[저장]** 버튼을 클릭합니다. 저장된 MCP는 **[내 MCP 목록]** 탭에서 확인할 수 있습니다.

    <Frame caption="나만의 MCP 만들기 — 이름 & 저장">
      <img src="/images/MCP_나만의 MCP 만들기(이름&저장).png" alt="이름 및 저장 화면" />
    </Frame>
  </Step>

  <Step title="조합 테스트">
    저장 후 우측의 **실시간 미리보기** 화면에서 답변을 받을 **LLM 모델을 선택**하고, 프롬프트를 입력하여 MCP를 테스트할 수 있습니다. 테스트 실행 시 아래 4단계 프로세스가 순서대로 진행됩니다.

    | 단계 | 영문 표기 | 설명 |
    |------|----------|------|
    | 1 | **prompt_sent** | 사용자 메시지와 도구 목록이 AI에게 전달 |
    | 2 | **tool_decision** | AI가 필요한 도구를 자율적으로 판단 |
    | 3 | **tool_execution** | 선택된 도구가 외부 서버에서 실행 |
    | 4 | **response_generated** | 실행 결과를 자연어 답변으로 생성 |

    <Frame caption="나만의 MCP 만들기 — 테스트 실행">
      <img src="/images/MCP_나만의 MCP 만들기(테스트).png" alt="MCP 테스트 화면" />
    </Frame>
  </Step>

  <Step title="결과 확인">
    MCP가 도구를 활용하여 생성한 결과를 확인합니다.

    <Frame caption="나만의 MCP 만들기 — 실행 결과">
      <img src="/images/MCP_나만의 MCP 만들기(결과).png" alt="MCP 실행 결과" />
    </Frame>
  </Step>
</Steps>

<Tip>
결과를 확인한 후 MCP 상태를 **OFF로 변경**하면 MCP가 잘 작동되는지 비교 확인할 수 있습니다.
</Tip>

## 내 MCP 목록 관리

생성된 MCP는 **[내 MCP 목록]** 탭에서 확인하고 관리할 수 있습니다. 각 MCP 카드 우측 상단의 **`⋮` 버튼**을 클릭하면 관리 메뉴가 나타납니다.

<Frame caption="내 MCP 목록 화면">
  <img src="/images/MCP_ 내 MCP 목록.png" alt="내 MCP 목록" />
</Frame>

| 기능 | 설명 |
|------|------|
| **활성/비활성** | 비활성화 시 에이전트 빌더 노드에서 해당 MCP를 활용할 수 없습니다 |
| **공유** | 클래스에 내가 만든 MCP를 공유합니다 |
| **테스트** | 만든 MCP를 테스트합니다 |
| **편집** | MCP를 다시 편집합니다 |
| **삭제** | MCP를 삭제합니다 |

## 수업 진행 팁

<Tip>
