# Step 3: 지식베이스 실습

## 간편 업로드

가장 빠르게 지식베이스를 활용하는 방법입니다.

<Steps>
  <Step title="지식베이스 메뉴 이동">
    사이드바에서 **[지식베이스]** 메뉴를 클릭합니다. 생성 방식 선택 화면이 나타납니다.

    <Frame caption="지식베이스 생성 방식 선택 화면">
      <img src="/images/지식베이스 화면(1).png" alt="지식베이스 생성 방식 선택" />
    </Frame>
  </Step>

  <Step title="간편 업로드 선택">
    **[간편 업로드]** 버튼을 선택합니다.
  </Step>

  <Step title="파일 업로드">
    파일을 **드래그하거나 클릭**하여 업로드합니다. 자동으로 임베딩이 진행됩니다.

    <Frame caption="간편 업로드 — 파일 업로드 화면">
      <img src="/images/파일 업로드_간편업로드.png" alt="파일 업로드 화면" />
    </Frame>
  </Step>

  <Step title="업로드 완료 및 목록 확인">
    업로드가 완료되면 **자동으로 지식베이스 목록 화면**으로 이동하여 등록된 파일을 바로 확인할 수 있습니다.
  </Step>

  <Step title="지식베이스 연결">
    AI 실습 화면에서 대화 입력창 좌측 **[+]** 버튼을 클릭합니다.
  </Step>

  <Step title="파일 선택">
    **[지식베이스 연결]**을 선택하고 참조할 파일을 체크한 후 **[연결하기]** 버튼을 클릭합니다.
  </Step>

  <Step title="질문 입력">
    질문을 입력하면 **문서 내용을 참조하여 답변**이 생성됩니다.
  </Step>
</Steps>

## 고급 설정

정밀한 검색 제어가 필요할 때 사용하는 설정 방식입니다.

<Steps>
  <Step title="지식베이스 메뉴 이동">
    사이드바에서 **[지식베이스]** 메뉴를 클릭합니다.
  </Step>

  <Step title="고급 설정 탭 선택">
    **[고급 설정]** 탭을 선택합니다.
  </Step>

  <Step title="파일 선택">
    파일을 선택한 후 **[다음 단계]**를 클릭합니다.
  </Step>

  <Step title="청킹 설정">
    청킹 방식을 선택합니다(**[일반]** 또는 **[Parents-Child]**). 세부 파라미터를 조정합니다(최대 길이 800, 중첩 200).

    <Frame caption="청킹 설정 화면">
      <img src="/images/청킹 설정.png" alt="청킹 설정" />
    </Frame>
  </Step>

  <Step title="프리뷰 청크">
    **[프리뷰 청크]**를 클릭하여 문서가 잘 나뉘었는지 확인합니다.
  </Step>

  <Step title="인덱스 모드 선택">
    **[고품질]** 임베딩 또는 **[경제적]** 키워드 방식 중 선택합니다.
  </Step>

  <Step title="검색 설정">
    리랭크 모델을 활성화하고, **Top-K / Top-N / 유사도 임계값**을 설정합니다.

    <Frame caption="검색 설정 화면">
      <img src="/images/검색 설정.png" alt="검색 설정" />
    </Frame>
  </Step>

  <Step title="최종 확인">
    설정 내용을 최종 확인한 후 **[지식베이스 생성]**을 클릭합니다.
  </Step>

  <Step title="생성 완료">
    생성 완료를 확인합니다.
  </Step>
</Steps>

<Tip>
생성 후 반드시 **테스트 질문**을 던져보세요. "문서 내용을 기반으로 답변해줘"라고 질문했을 때 답변 하단에 **참조 문서와 페이지가 정확히 표시**되는지 확인해야 합니다.
</Tip>

## 비교 모드 사용법

<Steps>
  <Step title="비교 모드 진입">
    지식베이스 메뉴에서 **'비교 모드'** 탭을 클릭합니다.
  </Step>

  <Step title="모델 선택">
    우측 상단에서 비교에 사용할 **LLM 모델**을 선택합니다.
  </Step>

  <Step title="모드 선택">
    각 패널 상단에서 **순수 LLM / 문서 참조 / RAG 설정** 중 선택합니다.

    <Frame caption="비교 모드 — 패널 A/B 화면">
      <img src="/images/지식베이스_비교 모드.png" alt="지식베이스 비교 모드" />
    </Frame>
  </Step>

  <Step title="파라미터 설정">
    비교할 대상 문서와 검색 파라미터를 각 패널에 **다르게 설정**합니다.
  </Step>

  <Step title="질문 및 비교">
    하단 입력창에 질문을 입력하면 **양쪽에 동시 전송**되어 결과를 바로 비교할 수 있습니다.
  </Step>
</Steps>

## 수업 진행 팁

<Tip>
- **수업 교재 업로드** — 수업 교재를 직접 업로드하여 RAG Q&A 실습을 하면 학생들의 이해도가 높아집니다.
- **청킹 비교** — 같은 문서를 다른 청킹 설정으로 업로드하여 검색 결과 차이를 비교해보세요.
- **RAG ON/OFF 비교** — 지식베이스 연결 전후의 답변 품질 차이를 비교하면 RAG의 효과를 체감할 수 있습니다.
</Tip>
