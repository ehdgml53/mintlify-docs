# 사용량 통계 & 설정

AI 모델 사용량을 분석하고, 강의별·학생별 통계를 확인하는 방법을 안내합니다.

---

## 사용량 통계 페이지 구성

<Accordion title="1. 기간 필터 탭">
  **일간 / 주간 / 월간 / 전체** 중 선택할 수 있습니다. 선택한 기간의 데이터만 집계되어 표시됩니다.
</Accordion>

<Accordion title="2. 강의 필터">
  오른쪽 상단 드롭다운에서 특정 강의를 선택하면 해당 강의 데이터만 필터링하여 확인할 수 있습니다.
</Accordion>

<Accordion title="3. 요약 카드">
  핵심 지표인 **총 대화 · 학생 수 · 활성 학생 · 학생당 평균 사용량**을 한눈에 확인할 수 있습니다.
</Accordion>

<Accordion title="4. 일별 막대 차트">
  날짜별 AI 대화 건수 추이를 시각화합니다. 차트를 드래그하여 특정 구간을 확대할 수 있습니다.
</Accordion>

<Accordion title="5. AI 모델별 사용현황">
  각 모델의 이름, 사용 횟수, 비중을 상세하게 표시합니다.
</Accordion>

---

## 기능별 사용 현황

| 기능 | 설명 |
|------|------|
| 비교 모드 | 여러 AI 모델을 동시에 비교하며 실습한 학생 수와 실습 횟수 집계 |
| 지식베이스 활용 | RAG 기능으로 문서를 업로드한 후 해당 내용을 기반으로 질문한 건수 |
| 프롬프트 (템플릿) | 학생들이 프롬프트를 템플릿으로 저장하고 재사용한 횟수 |

---

## 강의별 사용량

강의별 사용량 테이블의 각 컬럼에 대한 설명입니다.

| 컬럼 | 설명 |
|------|------|
| 강의명 | 등록된 강의 이름. 클릭하면 학생별 상세 사용량 펼쳐짐 |
| 학생 수 | 해당 강의에 등록된 총 학생 수 |
| 총 대화 | 강의 기간 내 발생한 모든 학생의 AI 대화 건수 합계 |
| 학생당 평균 | 총 대화 / 학생 수로 자동 계산 |
| 상태 | 진행 중 / 종료 |

---

## 학생별 사용량

### 참여도 기반 순위

총 대화 건수가 많은 순서대로 자동 정렬됩니다 (1등부터).

### 주요 데이터 컬럼

| 항목 | 설명 |
|------|------|
| 순위 | 메달 또는 숫자로 표시 |
| 학생명 | 아바타와 함께 표시 |
| 총 대화 | 정렬 기준이 되는 AI 대화 건수 |
| 최근 활동 일시 | 마지막 활동 시간 |

### 상세보기

우측 끝 버튼을 클릭하면 개별 학생의 학습 이력을 상세하게 조회할 수 있습니다.

<Tip>
  학생별 사용량을 정기적으로 확인하여 참여도가 낮은 학생을 조기에 파악하고 독려할 수 있습니다.
</Tip>
