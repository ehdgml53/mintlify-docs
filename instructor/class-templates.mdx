# 강의 템플릿

## 기능 소개

강의 템플릿은 같은 클래스에 소속된 **학생과 강사가 템플릿을 공유**하는 공간입니다. 프롬프트, 에이전트, 스킬 등 내가 만든 템플릿을 공유하면 클래스 전체가 활용할 수 있습니다.

- **공유**: 내가 만든 프롬프트/에이전트/스킬을 클래스에 공유
- **복사**: 다른 사람이 공유한 템플릿을 내 템플릿에 복사하여 바로 사용하거나 수정
- **코멘트**: 템플릿에 대한 피드백과 의견 교환

---

|----------|------|
| **프롬프트** | 시스템 프롬프트 템플릿 |

| **에이전트** | 에이전트 빌더 워크플로우 템플릿 |
| **스킬** | 스킬(skill.md) 템플릿 |

## 내 템플릿에 복사

상세 보기 화면에서 **[내 템플릿에 복사]** 버튼을 클릭하면 해당 템플릿이 내 실습 환경에 저장됩니다.

| 템플릿 유형 | 복사 위치 |
|------------|----------|
| **프롬프트** | LLM 실습 메뉴에서 확인 |
| **에이전트** | 에이전트 빌더 > 나의 에이전트에서 확인 |
| **스킬** | 에이전트 빌더 > 스킬 라이브러리에서 확인 |

복사된 템플릿은 **그대로 사용**하거나 **수정하여 나만의 버전으로 활용**할 수 있습니다.

---

## 템플릿 공유하기

내가 만든 프롬프트, 에이전트, 스킬을 클래스에 공유하려면 각 실습 메뉴에서 **공유** 기능을 사용하거나, 강의 템플릿 화면 우측 상단의 **[공유하기]** 버튼을 클릭합니다.

## 수업 진행 팁

<Tip>
- **예시 템플릿 먼저 공유** — 강사가 잘 만든 예시 템플릿을 먼저 공유하고, 학생들이 복사 후 수정하는 방식으로 진행하세요.
- **코멘트 활용** — 학생들이 서로의 템플릿에 코멘트를 달도록 유도하면 피드백 문화를 형성할 수 있습니다.
- **좋아요 랭킹** — 좋아요 정렬을 활용하여 우수 템플릿을 선정하면 학생들의 참여도가 높아집니다.
</Tip>
