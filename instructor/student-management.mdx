# 학생 관리

학생을 강의에 초대하고, 개별 학생의 학습 현황과 상세 리포트를 확인하는 방법을 안내합니다.

---

## 학생 초대하기

<Steps>
  <Step title="[학생 초대] 버튼 클릭">
    사이드바 메뉴에서 **[학생 관리]** 페이지로 이동한 후, 화면 우측 상단의 **[학생 초대]** 버튼을 클릭합니다.
  </Step>
  <Step title="강의 선택">
    초대 팝업창에서 초대할 강의를 선택합니다. 선택한 강의의 초대 코드가 하단에 표시됩니다.
  </Step>
  <Step title="초대 코드 공유">
    초대코드를 수강하는 학생들에게 공유합니다.
  </Step>
</Steps>

---

## 학생 상세 리포트

학생 목록에서 이름을 클릭하면 해당 학생의 상세 활동 데이터를 확인할 수 있습니다.

### 수강 정보

| 항목 | 설명 |
|------|------|
| 강의명 | 학생이 등록된 강의 이름 |
| 수강 기간 | 강의의 시작일과 종료일 |
| [수강 중] 배지 | 현재 수강 중인 강의에 표시. 종료 시 사라짐 |

### 활동 지표

| 지표 | 설명 |
|------|------|
| 총 로그인 | 플랫폼에 접속한 총 횟수. 학습 참여 의지 파악 |
| 활동 일수 | 실제 학습 활동이 발생한 날수. 실질적 학습 여부 |
| 마지막 접속일 | 가장 최근 접속일. 장기 미접속 학생 파악 |

### AI 실습 현황

| 항목 | 설명 |
|------|------|
| 프롬프트 수 | 저장하거나 사용한 프롬프트 템플릿 수 |
| 총 대화 수 | AI 모델과 나눈 전체 메시지 횟수. 적극적인 AI 활용도 |
| 사용 비용 / 평균 토큰 | 누적 API 비용과 대화 1건당 평균 토큰 수. 크레딧 소진 모니터링 |

### 학습 활동 요약

| 항목 | 설명 |
|------|------|
| 프로젝트 생성 | 에이전트·지식베이스 등을 묶어 관리하는 학습 단위를 만든 횟수 |
| 지식베이스 활용 | 문서를 업로드하고 RAG 검색 실습을 진행한 횟수 |
| MCP 도구 사용 | 에이전트 실행 중 외부 도구(웹 검색, 계산기 등)를 호출한 횟수 |
