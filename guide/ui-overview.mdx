## 사이드바 메뉴 상세 — 7개 핵심 메뉴

<AccordionGroup>
  <Accordion title="1. 내 프로젝트" icon="folder">
    주제별로 대화를 분류하고 관리합니다. 프로젝트를 생성, 편집, 삭제할 수 있으며, 수업 주제나 과제 단위로 대화를 체계적으로 정리할 수 있습니다.
  </Accordion>

  <Accordion title="2. LLM 실습" icon="comments">
    다양한 AI 모델과 직접 대화합니다. 최대 **3개 모델을 동시에 비교**하여 각 모델의 특성과 강점을 체험할 수 있습니다.
  </Accordion>

  <Accordion title="3. 프롬프트" icon="wand-magic-sparkles">
    시스템 프롬프트와 Few-shot 예시를 만들고 관리합니다. 효과적인 프롬프트 템플릿을 저장하여 반복 활용할 수 있습니다.
  </Accordion>

  <Accordion title="4. 지식베이스 (RAG)" icon="book">
    문서를 업로드하면 AI가 해당 문서를 참조하여 정확한 답변을 생성합니다. PDF, 텍스트 등 다양한 형식을 지원합니다.
  </Accordion>

  <Accordion title="5. MCP" icon="plug">
    웹검색, 계산기, 날씨 등 **10가지 외부 도구**를 조합하여 AI에 연결합니다. AI의 손과 발 역할을 하는 확장 기능입니다.
  </Accordion>

  <Accordion title="6. 에이전트 빌더" icon="diagram-project">
    노드 기반 워크플로우로 자율적으로 판단하고 행동하는 AI 에이전트를 설계합니다. AI의 두뇌 역할을 담당하는 핵심 기능입니다.
  </Accordion>

  <Accordion title="7. 학습 기록" icon="chart-line">
    AI 실습 사용량, 대화 기록, 에이전트 통계 등을 대시보드로 확인합니다. 학습 진행 상황을 한눈에 파악할 수 있습니다.
  </Accordion>
</AccordionGroup>

## 프로필 / 설정

사이드바 하단의 프로필 영역에서 다양한 설정을 변경할 수 있습니다.

| 설정 항목 | 설명 | 상태 |
|-----------|------|------|
| **계정 정보** | 이름, 이메일, 프로필 사진 변경 | 사용 가능 |
| **알림 설정** | 알림 수신 여부 및 방식 설정 | 준비중 |
| **비밀번호 변경** | 계정 비밀번호 재설정 | 사용 가능 |
| **수강 강의** | 현재 수강 중인 강의 목록 확인, 초대코드로 새 강의 등록 | 사용 가능 |
| **테마 설정** | 다크 모드 등 화면 테마 변경 | 사용 가능 |
| **언어 설정** | 인터페이스 언어 변경 | 준비중 |
