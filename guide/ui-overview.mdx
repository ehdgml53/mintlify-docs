## 훈련생 사이드바 메뉴

훈련생(Trainee)이 사용하는 핵심 메뉴입니다. 5단계 실습 과정과 학습 지원 기능으로 구성됩니다.

### 5단계 실습 메뉴

<AccordionGroup>
  <Accordion title="LLM 실습" icon="comments">
    **27개 이상의 AI 모델**과 직접 대화합니다. 최대 **3개 모델을 동시에 비교**하여 각 모델의 특성과 강점을 체험할 수 있습니다.

    - 단일 / 비교 모드 (최대 3개 모델 동시 비교)
    - 파라미터 조정 (Temperature, Top P, Max Tokens)
    - 스타일 프리셋 (정밀, 균형, 창의적)
    - 프롬프트/Few-shot 템플릿 연결 & 지식베이스 연결
  </Accordion>

  <Accordion title="프롬프트" icon="wand-magic-sparkles">
    시스템 프롬프트와 Few-shot 예시를 만들고 관리합니다. **AI 자동 생성 기능**으로 자연어를 입력하면 프롬프트와 Few-shot이 자동으로 생성됩니다.

    - AI 자동 생성 (프롬프트 / Few-shot)
    - 직접 작성 및 템플릿 저장
    - 활성화/비활성화로 LLM 실습 및 에이전트 빌더에서 바로 활용
    - 강의 템플릿으로 클래스 내 공유
  </Accordion>

  <Accordion title="지식베이스" icon="book">
    내 문서를 AI에 연결하면 조직 맥락을 이해하는 **전용 AI**를 만들 수 있습니다. 한번 등록하면 LLM 실습과 에이전트 빌더 양쪽에서 모두 활용 가능합니다.

    - 간편 업로드 (PDF, TXT, CSV, DOCX, .md)
    - 고급 설정 (청킹 크기, 오버랩, 임베딩 모델, 리랭킹)
    - RAG 비교 모드 (순수 LLM vs 문서 참조 vs RAG 설정)
    - 참조 소스 표시 및 할루시네이션 감소 효과 체험
  </Accordion>

  <Accordion title="MCP 도구" icon="plug">
    AI에게 "손과 발"을 달아주는 기능입니다. **총 105종의 외부 도구**를 제공하며, AI 자동 구성 기능으로 프롬프트 하나만 입력하면 필요한 도구를 자연스럽게 연결해줍니다.

    - 플랫폼 기본 도구 48종 (계산기, DART, 날씨, 환율, 차트, PDF/다이어그램 생성, 취업·진로 Work24 등)
    - 외부 서비스 연동 57종 (Notion, Slack, GitHub, Google Drive/Docs/Sheets/Slides, Gmail)
    - AI 자동 구성으로 도구 조합 설정
    
  </Accordion>

  <Accordion title="에이전트 빌더" icon="diagram-project">
    나 대신 일하는 AI 자동화 에이전트를 **노코드**로 설계합니다. 자연어 입력으로 에이전트를 자동 생성할 수 있으며, **에이전트 정보 + 8종 구성 블록**(KNOWLEDGE·TOOLBOX·SKILLS·SUB-AGENTS·FEW-SHOT·SCHEDULE·CHANNELS 등) 구조로 동작합니다.

    - 기본 노드: 시작/종료, 프롬프트, 스킬, 지식베이스, MCP 도구
    - 로직 노드: 분류, 조건 분기, 반복, 데이터 변환, 상태 설정, 승인, 가드레일
    - 멀티모달 노드: 비전 분석, 이미지 생성, OCR, 메모
    - AI 자동 생성 & 워크플로우 검증
  </Accordion>
</AccordionGroup>

### 학습 지원 메뉴

<AccordionGroup>
  <Accordion title="강의 템플릿" icon="share-nodes">
    강사가 배포하거나 같은 클래스의 수강생들이 서로 공유하는 **템플릿 공간**입니다. 불러와서 바로 내 실습에 적용할 수 있습니다.

    **공유 가능한 콘텐츠:**
    - 프롬프트 (시스템 프롬프트 템플릿)
    - Few-shot 예시 (입출력 예시 세트)
    - MCP 설정 (도구 조합 구성)
    - 에이전트 (워크플로우 에이전트)
    - 스킬 (AI 역할 및 행동 규칙 설정)
  </Accordion>

  <Accordion title="내 프로젝트" icon="folder">
    주제별로 대화를 분류하고 관리합니다. 수업 주제나 과제 단위로 대화를 체계적으로 정리할 수 있습니다.
  </Accordion>

  <Accordion title="학습 기록" icon="chart-line">
    5개 학습 영역(AI 실습, 지식베이스, 프롬프트, MCP, 에이전트 빌더)의 성과를 대시보드로 확인합니다.
  </Accordion>
</AccordionGroup>

## 프로필 / 설정

사이드바 하단의 프로필 영역에서 다양한 설정을 변경할 수 있습니다.

| 설정 항목 | 설명 |
|-----------|------|
| **계정 정보** | 이름, 이메일, 프로필 사진 변경 |
| **비밀번호 변경** | 계정 비밀번호 재설정 |
| **수강 강의** | 현재 수강 중인 강의 목록 확인, 초대코드로 새 강의 등록 |
| **테마 설정** | 다크 모드 등 화면 테마 변경 |

## 시작하기 핵심 체크리스트

<Tip>
다음 항목을 확인하여 GrowFit 사용 준비를 마무리하세요.

- [ ] **사이드바 메뉴 익히기** — 핵심 메뉴의 위치와 역할을 파악합니다.
- [ ] **프로필 설정 확인** — 계정 정보와 수강 강의가 올바르게 등록되었는지 확인합니다.
- [ ] **강의 템플릿 확인** — 강사가 배포한 템플릿이 있는지 확인합니다.
- [ ] **다음 단계 진행** — 5단계 AI 실습(LLM → 프롬프트 → 지식베이스 → MCP → 에이전트 빌더)을 시작합니다.
</Tip>
