# 자주 묻는 질문 (FAQ)

<AccordionGroup>
  <Accordion title="어떤 모델을 선택해야 할까요?">
    작업의 목적과 복잡도에 따라 가장 적합한 모델을 선택하세요.

    | 상황 | 추천 모델 | 적합한 작업 |
    |------|----------|------------|
    | **빠른 응답이 필요할 때** | GPT-4o mini, Claude 3.5 Haiku | 간단한 질문, 아이디어 브레인스토밍, 빠른 초안 작성 |
    | **복잡한 분석이 필요할 때** | GPT-4o | 논리적 추론, 코딩, 데이터 분석, 창의적인 글쓰기 |

    <Tip>
    어떤 모델이 좋은지 확신이 서지 않을 때는 **비교 모드**를 활용하여 2~3개 모델의 답변을 동시에 받아 비교해보세요.
    </Tip>
  </Accordion>

  <Accordion title="한 번에 몇 개의 모델까지 비교할 수 있나요?">
    **최대 3개의 모델**을 동시에 선택하여 답변을 비교할 수 있습니다.

    - **2개 이상**의 모델을 선택하면 자동으로 비교 모드가 활성화되어 화면이 분할됩니다.
    - **1개**의 모델만 선택하면 즉시 일반 대화 모드(단일 모드)로 전환됩니다.
  </Accordion>

  <Accordion title="초대코드를 받지 못했어요.">
    강사님께 직접 문의하시거나, 아래 체크사항을 확인해보세요.

    - **강사 및 관리자 문의**: 초대코드는 강사가 강의를 생성한 후 발급됩니다. 수업 담당자에게 직접 문의하세요.
    - **이메일 스팸함 확인**: 스팸 메일함이나 프로모션 탭으로 분류되지 않았는지 확인하세요.
    - **입력 시 주의사항**: 코드는 대소문자를 정확히 구분합니다. 복사/붙여넣기 시 앞뒤 공백에 주의하세요.
  </Accordion>

  <Accordion title="이미지 분석/생성 기능은 언제 사용할 수 있나요?">
    멀티모달 기능은 **출시 예정**입니다. 출시 전까지는 텍스트 기반 기능을 먼저 충분히 익혀주세요.

    | 기능 | 설명 | 상태 |
    |------|------|------|
    | **비전 분석 노드** | 이미지의 내용을 AI가 분석하고 이해하여 텍스트로 설명 | 출시 예정 |
    | **이미지 생성 노드** | 텍스트 프롬프트를 입력하여 새로운 이미지 생성 | 출시 예정 |
    | **OCR 추출 노드** | 문서나 이미지 속 텍스트를 인식하여 디지털 텍스트로 변환 | 출시 예정 |
  </Accordion>

  <Accordion title="만든 에이전트를 학생들과 공유할 수 있나요?">
    네, 가능합니다. 공유 옵션을 활성화하고 사용량 및 통계를 체계적으로 관리하세요.

    - **공유 및 배포**: 에이전트를 'Published' 상태로 변경한 후 공유 옵션을 켜서 학생들에게 링크를 배포합니다.
    - **통계 모니터링**: 학생들이 에이전트를 얼마나 자주 사용하는지, 어떤 모델을 자주 사용하는지 실시간 데이터를 확인할 수 있습니다.
  </Accordion>
</AccordionGroup>
