# 자주 묻는 질문 (FAQ)

<AccordionGroup>
  <Accordion title="ChatGPT 쓸 줄 아는데 왜 GrowFit이 필요한가요?">
    ChatGPT에서 질문하는 건 AI 활용의 **10%**입니다. 내 데이터를 연결하고(RAG), 외부 도구를 붙이고(MCP), 자동화 시스템을 설계하는 것(에이전트 빌더)이 **나머지 90%**입니다.

    GrowFit은 이 90%를 한 공간에서 이론과 함께 **실습으로** 가르칩니다.
  </Accordion>

  <Accordion title="코딩을 못해도 할 수 있나요?">
    네, GrowFit은 **100% 노코드 플랫폼**입니다. 프롬프트, MCP 도구 조합, 에이전트 빌더 모두 AI 자동 생성 기능을 지원하여 자연어로 설명하면 AI가 알아서 만들어줍니다.

    비개발 직군도 자기 업무에 맞는 AI 시스템을 만들 수 있습니다.
  </Accordion>

  <Accordion title="어떤 모델을 선택해야 할까요?">
    작업의 목적과 복잡도에 따라 가장 적합한 모델을 선택하세요.

    | 상황 | 추천 모델 |
    |------|----------|
    | **빠른 응답이 필요할 때** | GPT-4o mini, Claude 3 Haiku, Gemini Flash |
    | **복잡한 분석이 필요할 때** | GPT-5, Claude Sonnet 4 |
    | **한국어 특화 작업** | K-EXAONE 236B |
    | **코딩/개발** | Qwen3 Coder, Devstral 2 |

    <Tip>
    어떤 모델이 좋은지 확신이 서지 않을 때는 **비교 모드**를 활용하여 최대 3개 모델의 답변을 동시에 비교해보세요.
    </Tip>
  </Accordion>

  <Accordion title="교육 시간은 어떻게 되나요?">
    **4시간 핵심 과정**(반나절)과 **8시간 심화 과정**(1일)을 기본으로 제공합니다.

    - **4시간 과정**: AI 비교 체험 & 프롬프트 설계 → 지식베이스(RAG) 실습
    - **8시간 과정**: 5단계 전체 실습 + 팀 프로젝트 → 실전 에이전트 제작 및 발표

    기업 맞춤 일정도 가능합니다.
  </Accordion>

  <Accordion title="다른 AI 교육 플랫폼과 뭐가 다른가요?">
    **이론 수업과 AI 실습을 한 공간에서 동시에** 진행할 수 있는 플랫폼은 GrowFit이 유일합니다.

    - 강사가 설명하면서 학생이 바로 같은 화면에서 실습
    - 27+ AI 모델 동시 비교 실습
    - 강의 템플릿으로 클래스 내 프롬프트, 에이전트 공유
    - 83종 MCP 도구로 실무 연결
  </Accordion>

  <Accordion title="한 번에 몇 개의 모델까지 비교할 수 있나요?">
    **최대 3개의 모델**을 동시에 선택하여 답변을 비교할 수 있습니다.

    - **2개 이상**의 모델을 선택하면 자동으로 비교 모드가 활성화되어 화면이 분할됩니다.
    - **1개**의 모델만 선택하면 즉시 일반 대화 모드(단일 모드)로 전환됩니다.
  </Accordion>

  <Accordion title="초대코드를 받지 못했어요.">
    강사님께 직접 문의하시거나, 아래 체크사항을 확인해보세요.

    - **강사 및 관리자 문의**: 초대코드는 강사가 강의를 생성한 후 발급됩니다.
    - **이메일 스팸함 확인**: 스팸 메일함이나 프로모션 탭으로 분류되지 않았는지 확인하세요.
    - **입력 시 주의사항**: 코드는 대소문자를 정확히 구분합니다. 복사/붙여넣기 시 앞뒤 공백에 주의하세요.
  </Accordion>

  <Accordion title="만든 에이전트를 학생들과 공유할 수 있나요?">
    네, 가능합니다. 강의 템플릿을 통해 프롬프트, Few-shot, MCP 설정, 에이전트를 클래스 내에서 공유할 수 있습니다.

    - **강사 배포**: 수업에 맞춰 준비한 템플릿을 수강생에게 배포
    - **수강생 공유**: 같은 클래스 수강생끼리 자신이 만든 템플릿을 서로 공유
    - **통계 모니터링**: 사용 현황을 실시간으로 확인
  </Accordion>
