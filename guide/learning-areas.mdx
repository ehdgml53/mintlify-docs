# 학습 영역 & 제공 모델

GrowFit은 AI를 체계적으로 학습할 수 있도록 **5단계 실습 체계**를 제공하며, **22종 이상의 AI 모델**을 통해 실습할 수 있습니다.

---

## 5단계 실습 체계

각 단계의 결과물이 다음 단계의 입력이 되는 **연결형 학습 설계**입니다.

<CardGroup cols={2}>
  <Card title="Step 1. LLM 실습" icon="comments">
    22개 이상의 AI 모델을 직접 비교하며, 상황에 맞는 최적의 모델을 선택하는 안목을 기릅니다.
  </Card>
  <Card title="Step 2. 프롬프트" icon="wand-magic-sparkles">
    AI 자동 생성 기능으로 시스템 프롬프트와 Few-shot을 작성하고, 답변 품질을 극대화합니다.
  </Card>
  <Card title="Step 3. 지식베이스" icon="book">
    내 문서를 AI에 연결하면 조직 맥락을 이해하는 전용 AI를 만들 수 있습니다.
  </Card>
  <Card title="Step 4. MCP 도구" icon="plug">
    70종 이상의 외부 도구를 AI에 연결하여 "말만 하는 AI"를 "진짜 일하는 AI"로 만듭니다.
  </Card>
  <Card title="Step 5. 에이전트 빌더" icon="diagram-project">
    16종 노드를 활용하여 나 대신 일하는 AI 자동화 시스템을 노코드로 설계합니다.
  </Card>
</CardGroup>

## 22+ 모델 라인업 & 선정 철학

### 모델 선정 철학

<Note>
**"어떤 AI가 최고인가?"** 가 아니라 **"이 작업에는 어떤 AI가 적합한가?"** 를 판단하는 능력을 기르는 것이 GrowFit의 목표입니다.
</Note>

GrowFit은 **범용 모델**과 **코딩 특화 모델**, **추론 특화 모델**을 균형 있게 제공하여 학습자가 작업 유형에 따라 최적의 모델을 선택하는 안목을 기를 수 있도록 설계되었습니다.

### 지원 모델

| 계열 | 모델 |
|------|------|
| OpenAI | GPT-5, GPT-5 mini, GPT-4.1, GPT-4o mini |
| Anthropic | Claude Sonnet 4, Claude 3 Haiku |
| Google | Gemini 2.5 Flash |
| 한국어 특화 | K-EXAONE 236B, Solar Pro 3 |
| 오픈소스 | Qwen3 235B, DeepSeek-R1, Llama 3.3 70B |
| 기타 | +10 more |

### 작업 유형별 추천 모델 가이드

| 작업 유형 | 추천 모델 | 추천 이유 |
|-----------|-----------|-----------|
| 코딩 / 개발 | Qwen3 Coder, Devstral 2, Claude | 코딩 특화 모델로 코드 생성, 디버깅, 리팩토링 최적화 |
| 창의적 글쓰기 | GPT, Gemini, Llama 70B | 창의성, 풍부한 어휘력, 다양한 표현 스타일 |
| 긴 문서 분석 | Claude | Claude 200K, Gemini 1M 토큰의 긴 문맥 처리 능력 |
| 빠른 응답 필요 | Gemini Flash, GPT-4o mini, Nemotron Nano | 빠른 응답 속도, 경량 모델 포함 |
| 한국어 특화 | K-EXAONE, Solar Pro 3 | 한국 기업이 만든 모델로 한국어 번역, 요약, 문서 분석에 강점 |
| 비용 효율 실습 | 무료 모델 (Llama, Qwen, Solar 등) | 무료 모델로 비용 부담 없이 수십 번 반복 실험 가능 |

<Note>
**특화 모델 참고:**
- **추론 특화:** DeepSeek R1, Qwen3 Thinking
- **이미지 분석:** Qwen3 VL, Mistral Small
- **이미지 생성:** Gemini Flash Image
</Note>

<Tip>
**모델 지속 확장** — 교육 적합성, 비용 효율성, 응답 품질을 기준으로 테스트 후 순차 추가됩니다.
</Tip>
