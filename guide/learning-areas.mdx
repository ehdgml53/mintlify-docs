# 학습 영역 & 제공 모델

GrowFit은 AI를 체계적으로 학습할 수 있도록 **6가지 핵심 영역**을 제공하며, **19종의 AI 모델**을 통해 실습할 수 있습니다.

---

## 6가지 학습 영역

<CardGroup cols={2}>
  <Card title="1. LLM 활용 (AI 실습)" icon="comments">
    다양한 AI 모델 비교 실습, Temperature·Top-P 등 파라미터 튜닝, 에이전트 선택으로 AI 대화를 진행합니다.
  </Card>
  <Card title="2. 프롬프트 엔지니어링" icon="wand-magic-sparkles">
    시스템 프롬프트, Few-shot 예시, 에이전트 템플릿 설계로 AI의 잠재 성능을 최대로 활용합니다.
  </Card>
  <Card title="3. 지식베이스" icon="book">
    문서 업로드 후 AI가 참조하여 정확하고 신뢰할 수 있는 답변을 생성합니다.
  </Card>
  <Card title="4. MCP" icon="plug">
    다양한 외부 도구를 AI에 연결하는 표준 프로토콜(Model Context Protocol)입니다.
  </Card>
  <Card title="5. 에이전트 빌더" icon="diagram-project">
    프롬프트·스킬·지식베이스·MCP를 노드로 배치하고 연결하여 AI 워크플로우를 설계합니다.
  </Card>
  <Card title="6. 프로젝트 관리" icon="folder-open">
    대화를 프로젝트 단위로 분류하고 체계적으로 관리합니다.
  </Card>
</CardGroup>

## 19종 모델 라인업 & 선정 철학

### 모델 선정 철학

<Note>
**"어떤 AI가 최고인가?"** 가 아니라 **"이 작업에는 어떤 AI가 적합한가?"** 를 판단하는 능력을 기르는 것이 GrowFit의 목표입니다.
</Note>

GrowFit은 **범용 모델**과 **코딩 특화 모델**, **추론 특화 모델**을 균형 있게 제공하여 학습자가 작업 유형에 따라 최적의 모델을 선택하는 안목을 기를 수 있도록 설계되었습니다.

---

### 작업 유형별 추천 모델 가이드

| 작업 유형 | 추천 모델 | 추천 이유 |
|-----------|-----------|-----------|
| 코딩 / 개발 | Qwen3 Coder, Devstral 2, Claude | 코딩 특화 모델로 코드 생성, 디버깅, 리팩토링 최적화 |
| 창의적 글쓰기 | GPT, Gemini, Llama 70B | 창의성, 풍부한 어휘력, 다양한 표현 스타일 |
| 긴 문서 분석 | Claude | Claude 200K, Gemini 1M 토큰의 긴 문맥 처리 능력 |
| 빠른 응답 필요 | Gemini Flash, GPT-3.5, Nemotron Nano | 빠른 응답 속도, 무료 경량 모델 포함 |
| 한국어 특화 | EXAONE, Solar Pro 3 | 한국 기업이 만든 모델로 한국어 번역, 요약, 문서 분석에 강점 |
| 비용 효율 실습 | 무료 14종 (Llama, Qwen, Solar 등) | 무료 모델로 비용 부담 없이 수십 번 반복 실험 가능 |

<Note>
**특화 모델 참고:**
- **추론 특화:** DeepSeek R1, Qwen3 Thinking
- **이미지 분석:** Qwen3 VL, Mistral Small
- **이미지 생성:** Gemini Flash Image
</Note>

<Tip>
**추가 모델 확장 계획** — 각 모델은 교육 적합성, 비용 효율성, 응답 품질 등을 기준으로 테스트를 거쳐 순차적으로 추가 예정입니다.
</Tip>
