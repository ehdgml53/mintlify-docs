# 프로젝트 관리

## 기능 소개

프로젝트는 AI 대화를 **주제별로 분류하고 체계적으로 관리**할 수 있는 기능입니다. 수업 주제, 과제, 업무 단위로 프로젝트를 만들어 관련 대화를 한 곳에 모아두면 나중에 학습 내용을 쉽게 찾고 복습할 수 있습니다.

## 새 프로젝트 생성

<Steps>
  <Step title="새 프로젝트 클릭">
    프로젝트 화면 우측 상단의 **[+ 새 프로젝트]** 버튼을 클릭합니다.
  </Step>

  <Step title="이름 & 설명 입력">
    **프로젝트 이름**(필수, 최대 50자)과 **프로젝트 설명**(선택)을 입력한 후 **[프로젝트 생성]** 버튼을 클릭합니다.

    <Frame caption="새 프로젝트 만들기">
      <img src="/images/프로젝트_ 새 프로젝트 생성.png" alt="새 프로젝트 생성 모달" />
    </Frame>
  </Step>
</Steps>

---

## 프로젝트 내부 화면

프로젝트를 클릭하면 해당 프로젝트에 포함된 **대화 목록**을 확인할 수 있습니다. 우측 상단에서 다양한 관리 기능을 사용할 수 있습니다.

<Frame caption="프로젝트 내부 — 대화 목록 및 관리 버튼">
  <img src="/images/프로젝트_ 생성 안 대화 .png" alt="프로젝트 내부 대화 목록" />
</Frame>

| 버튼 | 기능 |
|------|------|
| **새 대화** | 해당 프로젝트에 새로운 AI 대화를 시작합니다 |
| **수정** (편집 아이콘) | 프로젝트 이름과 설명을 수정합니다 |
| **삭제** (휴지통 아이콘) | 프로젝트를 삭제합니다 |
| **활성/비활성 토글** | 프로젝트를 활성 또는 비활성 상태로 전환합니다 |

<Frame caption="프로젝트 수정 — 이름/설명 변경">
  <img src="/images/프로젝트_수정버튼.png" alt="프로젝트 수정 모달" />
</Frame>

## 대화에 프로젝트 연결하기

기존 대화를 프로젝트에 연결하는 방법은 두 가지가 있습니다.

<Tabs>
  <Tab title="방법 1: AI 실습 화면에서">
    <Steps>
      <Step title="[+] 버튼 클릭">
        AI 실습 화면 하단의 입력창에서 **[+] 버튼**을 클릭합니다.
      </Step>
      <Step title="프로젝트 목록 선택">
        나타나는 메뉴에서 **[프로젝트 목록]** 버튼을 클릭합니다.
      </Step>
      <Step title="프로젝트 지정">
        목록에서 설정할 프로젝트를 선택합니다.
      </Step>
    </Steps>

    <Note>
    프로젝트는 **프로젝트 메뉴에서 미리 생성**해야 선택할 수 있습니다.
    </Note>
  </Tab>

  <Tab title="방법 2: 최근 대화 목록에서">
    <Steps>
      <Step title="대화 찾기">
        왼쪽 사이드바의 대화 목록에서 해당 대화를 찾습니다. 프로젝트에 연결된 대화는 프로젝트 태그가 표시됩니다.

        <Frame caption="대화 목록 — 프로젝트 태그 표시">
          <img src="/images/프로젝트_대화 목록(프로젝트 포함 대화).png" alt="프로젝트 포함 대화 목록" />
        </Frame>
      </Step>
      <Step title="옵션 메뉴 열기">
        대화 옆의 **[⋮] 버튼**을 클릭하고 **[프로젝트 추가]**를 선택합니다.

        <Frame caption="대화 옵션 메뉴 — 프로젝트 추가">
          <img src="/images/프로젝트_ 최근 대화 목록에서 추가 방법(1).png" alt="대화 옵션 메뉴" />
        </Frame>
      </Step>
      <Step title="프로젝트 선택">
        프로젝트 목록에서 원하는 프로젝트를 선택한 후 **[추가하기]** 버튼을 클릭합니다.

        <Frame caption="프로젝트에 추가 — 프로젝트 선택">
          <img src="/images/프로젝트_ 최근 대화 목록에서 추가 방법(2).png" alt="프로젝트에 추가 모달" />
        </Frame>
      </Step>
    </Steps>
  </Tab>
</Tabs>

## 프로젝트 관리 기능

프로젝트 목록을 효율적으로 관리할 수 있는 다양한 기능을 제공합니다.

| 기능 | 설명 |
|------|------|
| **검색** | 프로젝트 이름, 설명에 포함된 키워드로 즉시 검색 |
| **필터** | 전체 / 활성 프로젝트 / 비활성 프로젝트 상태별 필터링 |
| **정렬** | 최근 수정순, 이름순(가나다), 생성일순, 대화 수 기준 |
| **보기** | 그리드 보기 ↔ 목록 보기 전환 |

</Warning>

## 강사 TIP

<Tip>
프로젝트를 효과적으로 활용하는 방법:

- **주제별 생성** — 대화 내용을 쉽게 검색할 수 있습니다.
- **과제별 관리** — 학생 과제 이력을 체계적으로 모니터링할 수 있습니다.
- **완료 시 비활성화** — 진행 중인 프로젝트 목록을 깔끔하게 유지할 수 있습니다.
</Tip>
