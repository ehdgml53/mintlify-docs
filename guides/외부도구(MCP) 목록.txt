외부도구(MCP) 목록
GrowFit AI 외부 도구(MCP) 서비스 안내

총 89개 AI 도구 / 12개 카테고리 / 2026년 1월 서비스 발표 (2026년 4월 업데이트)

GrowFit은 AI 기반 플랫폼으로, 자연어로 질문하면 AI가 최적의 외부 도구를 자동으로 선택하여 실행하고, 그 결과를 답변에 반영합니다.

별도의 조작 없이 평소처럼 대화하면 됩니다. AI가 질문의 의도를 파악하여 계산, 검색, 데이터 조회 등 필요한 도구를 스스로 호출합니다.

총 도구 수 : 89개
카테고리 : 12개
빌트인 도구 : 45개 (API 키 불필요 13개 포함)
1차 서비스 개시 : 2026년 2월 (기본 유틸리티 13종)
2차 확장 : 2026년 3월 (공공데이터·생활정보·외부 연동 70종)
3차 확장 : 2026년 4월 (취업·진로 Work24 API 6종)

4. 우수 활용 사례 (도구 조합)
========================================

GrowFit AI는 하나의 대화에서 여러 도구를 조합하여 종합적인 답변을 제공합니다.

다음은 실제 학습 상황에서 도구 조합이 활용되고 있는 사례입니다.

----------------------------------------

사례 1: 해외 여행 준비

사용 도구: 날씨 + 환율 조회 + 공항 운항

----------------------------------------

사용자: "다음 주에 일본 여행 가는데, 도쿄 날씨랑 환율 알려줘"

AI: 도쿄 현재 날씨입니다.

    구름 조금 22도C (체감 21도C), 습도 55%

    현재 환율: 1 JPY = 9.12 KRW

    10,000엔 = 약 91,200원입니다.

사용자: "인천에서 도쿄행 비행기도 찾아줘"

AI: 인천 -> 도쿄 출발편 3건입니다.

    1. KE001 대한항공 -> 나리타 08:30 (Gate A12)

    2. OZ105 아시아나 -> 하네다 10:00 (Gate B05)

    3. 7C1101 제주항공 -> 나리타 14:20 (Gate C08)

-> AI가 대화 맥락을 파악하여 날씨, 환율, 항공편 도구를 순차적으로 호출합니다.

----------------------------------------

사례 2: 취업 준비 종합 리서치

사용 도구: 채용공고 검색 + 직업/학과 정보 + 국가자격 시험

----------------------------------------

사용자: "데이터분석가는 어떤 직업이야?"

AI: 데이터분석가 직업 정보입니다.

    급여: 3,500~6,000만원 / 전망: 매우 밝음

    관련학과: 통계학, 컴퓨터공학, 경영학

사용자: "그러면 관련 채용공고도 찾아줘"

AI: 데이터분석가 관련 채용공고 5건입니다.

    1. [네이버] 데이터 분석가 (정규직) - 경기 성남시

    2. [쿠팡] 비즈니스 인텔리전스 (정규직) - 서울 송파구

사용자: "빅데이터분석기사 시험 일정도 알려줘"

AI: 2026년 빅데이터분석기사 시험 일정입니다.

    필기접수: 2026.06.10~06.14

    필기시험: 2026.07.05~07.26

    합격발표: 2026.08.13

-> 직업 탐색에서 채용 시장 조사, 자격증 준비까지 하나의 대화에서 완결됩니다.

----------------------------------------

사례 3: 학교생활 도우미

사용 도구: 학교 시간표 + 학교 급식 + 도서관 정보

----------------------------------------

사용자: "서울중학교 2학년 3반 내일 시간표 알려줘"

    5교시 체육 / 6교시 음악

AI: 서울중학교 2026-04-14 중식 메뉴입니다.
